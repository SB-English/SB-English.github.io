日本很多企业开始裁员，以应对新冠和日本同工同酬的劳动政策带来的潜在风险
Created2021-03-14|Updated2021-03-14|就业动态|解雇
日本很多企业，包括世界500强企业近期都在裁员！主要表现在：1，同等职位中优先裁掉工资较高的，以降低同职位的平均工资，以实现降低新员工入职的工资水平。2，为符合同工同酬，以实现派遣用工、临时用工的可持续性，降低正社员的工资水平，以实现正社员的成本和派遣用工的相近工资水平，规避法律风险。3，进入2021年3月，一些派遣公司的招聘广告的年收标准较之前降低了100万日元以上。

Created2001-10-27|Updated2020-07-18|GlobalChinaLow|中国•法律•劳动法•工会
